# Simple JPG to PDF Converter (画像PDF変換ツール)

複数のJPEG/JPG画像を、一瞬で1つのPDFファイルに結合・変換する軽量なデスクトップアプリです。`img2pdf`ライブラリを使用しているため、画像の画質を一切落とさずに変換が可能です。

## 🌟 特徴
* **画質の劣化なし:** 画像を再エンコードしないため、元の画質を100%維持したままPDF化します。
* **超高速:** 変換処理は一瞬で完了します。
* **まとめて変換:** 複数の画像を選択して、1つのPDFに結合できます。
* **インストール不要:** EXEファイルをダウンロードするだけで、すぐに使えます。

## 💻 動作環境
* Windows 10 / 11
* （PowerPointやAcrobatなどの外部ソフトは不要です）

## 🚀 ダウンロード方法
1.  [Releases](https://github.com/[あなたのユーザー名]/[リポジトリ名]/releases) ページにアクセスします。
2.  最新バージョンの `jpg2pdf.exe` をダウンロードします。
3.  ダウンロードしたファイルをダブルクリックして起動してください。

## 🛠️ 使い方
1.  **「ファイル選択」** ボタンをクリックします。
2.  PDFにしたい画像（.jpg または .jpeg）をすべて選択します。
3.  **「実行」** をクリックします。
4.  変換された `output.pdf` が **ダウンロード** フォルダに保存されます。
5.  保存場所を開くかどうかの確認ダイアログが表示されます。
